# Android app

Inserire qui il progetto Android `EmergencyMobile`.

Prima del push pubblico, verificare che non siano presenti:

- `local.properties`
- cartelle `build/`
- cartelle `.gradle/`
- keystore
- credenziali
- endpoint reali usati solo per demo privata

Il file `ApiConstants.example.kt` contiene i placeholder da sostituire con gli output CloudFormation.
